# CV HTML Implementation Plan

> **Goal:** Reproduce Kaiser Styve CV from Figma/PDF as editable HTML/CSS + JSON experiences.

**Architecture:** Single-page A4 layout, 36/64 two-column grid, CSS tokens from PDF color extraction, `render.js` hydrates experiences from JSON.

**Tech Stack:** HTML5, CSS3 (tokens/layout/print), vanilla JS, Google Fonts (Poppins).
